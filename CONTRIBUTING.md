## How to Contribute
1. **Report Bugs:** Open an issue with a clear description and steps to reproduce.
2. **Suggest Features:** Use the "Discussions" or "Issues" tab.
3. **Submit Code:** - Fork the repo.
   - Create a feature branch.
   - Ensure your code follows our style guidelines.
   - Submit a PR with a description of your changes.

**Note:** We do not accept contributions that include AI-generated code unless you have manually reviewed, verified, and significantly modified it to fit the project architecture.
